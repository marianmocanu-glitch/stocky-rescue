🛟 Stocky Rescue

Stocky shuts down on August 31, 2026. Shopify's own migration guide says "Suppliers can't be exported from Stocky." They can — through Stocky's API. This free tool downloads your suppliers, full purchase-order history and stock-adjustment history as clean CSV files. Everything runs in your browser; your API key is never stored.

Where do I find my API key?
  1. Open Stocky from your Shopify admin (Apps → Stocky).
  2. Go to Preferences → API (stocky.shopifyapps.com/preferences/api).
  3. Copy the API key shown there and paste it above.

The key only allows reading your Stocky data. Revoke it in the same screen whenever you like.

🔒 Your key is used only during the export, relayed straight to Stocky, and never stored or logged. Read the code.

    FAQ

    Is this affiliated with Shopify?
    No. It's an independent, free, open-source tool.
    What exactly is exported?
    suppliers.csv, purchase_orders.csv, purchase_order_line_items.csv, stock_adjustments.csv, stock_adjustment_items.csv, tax_types.csv + a README.
    What is NOT exported?
    Anything Stocky's API doesn't expose — e.g. supplier lead times/MOQs aren't available through the API. Reports you can already export from Stocky's UI aren't duplicated here.
    Why free?
    Because losing your supplier records to an app shutdown is unfair, and because we hope you'll try our paid replacement app later. No strings attached.
